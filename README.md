# MVPStruct

最近在学习MVP设计模式，所有就写了个小程序来实践下，纯属娱乐
项目数据均来自[聚合数据](https://www.juhe.cn/)

##项目架构
采用MVP+Retrofit

##实现功能
###1. 新闻模块
采用ViewPage + Fragment显示新闻列表，内容跳转用webview加载网页

###2. 开心一刻
本人平时喜欢看笑话，所以就自己做了一个

###3. 微信精选
获取最新的微信精选列表，采用RecyclerView显示数据
